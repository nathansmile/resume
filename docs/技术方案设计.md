# 简历智能筛选系统 - 技术方案设计

## 1. 项目概述

### 1.1 项目目标
构建一个全栈AI简历筛选系统，实现简历上传、智能解析、岗位匹配评分和候选人管理功能。

### 1.2 核心功能
- 批量简历上传与PDF解析
- AI智能信息提取（流式返回）
- 岗位匹配与智能评分
- 候选人管理面板
- 数据可视化与对比分析

## 2. 技术栈选型

### 2.1 前端技术栈
- **框架**: Vite 5+ + React 18 + AntDesign
- **语言**: TypeScript 5+
- **UI组件库**: shadcn/ui + Radix UI
- **样式方案**: Tailwind CSS
- **状态管理**: Zustand
- **数据请求**: TanStack Query (React Query)
- **表单处理**: React Hook Form + Zod
- **图表库**: Recharts
- **PDF预览**: react-pdf
- **文件上传**: react-dropzone
- **SSE客户端**: EventSource API
- **路由**: React Router v6

### 2.2 后端技术栈
- **框架**: NestJS 10+ (Node.js 20+)
- **语言**: TypeScript 5+
- **PDF解析**: pdf-parse / pdfjs-dist
- **AI集成**: Anthropic Claude API (模块化设计，支持切换)
- **数据库**: PostgreSQL 15+
- **ORM**: Prisma 5+
- **数据验证**: class-validator + class-transformer
- **缓存**: Redis + ioredis
- **文件存储**: 本地磁盘存储 (multer)
- **SSE**: NestJS 内置 SSE 支持

### 2.3 部署方案
- **前端**: Vercel (Vite SPA)
- **后端**: Railway / Render
- **数据库**: Railway PostgreSQL
- **文件存储**: 本地磁盘 (开发) / S3 (生产可选)

### 4.3 模块三：岗位匹配与智能评分

#### 评分算法设计
```python
# 评分端点
POST /api/evaluations/score

def calculate_match_score(candidate, job_description):
    # 1. 技能匹配度 (40%)
    skill_score = calculate_skill_match(
        candidate.skills,
        job_description.required_skills,
        job_description.preferred_skills
    )

    # 2. 经验相关性 (35%)
    experience_score = calculate_experience_match(
        candidate.work_experiences,
        job_description.description
    )

    # 3. 教育背景契合度 (25%)
    education_score = calculate_education_match(
        candidate.educations,
        job_description.education_requirements
    )

    # 综合评分
    overall_score = (
        skill_score * 0.4 +
        experience_score * 0.35 +
        education_score * 0.25
    )

    # 调用AI生成评语
    ai_comment = generate_ai_comment(candidate, job_description, scores)

    return {
        "overall_score": overall_score,

        "education_score": education_score,
        "ai_comment": ai_comment
    }
```
